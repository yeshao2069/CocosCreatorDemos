### 功能介绍
游戏操作键位：
- W=前进
- A=左移
- S=后退
- D=右移
- shift + W=加速跑
- C=下蹲
- 鼠标左键=射击
- 鼠标右键=瞄准
- 空格=跳跃
- Q=肘击
- R=换弹夹
- 鼠标滑轮=切枪
